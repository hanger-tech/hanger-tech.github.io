Ubuntu24.04でUSキーボードに変更する方法

コンソールをUSキーボードに変更

sudo dpkg-reconfigure keyboard-configuration

Generic 105-key PC -> Generic 102-key PC -> English (US) -> English (US) -> キーボード配置のデフォルト -> コンポーズキーなし -> いいえ

GUIでUSキーボードに変更

設定 -> キーボード 入力ソースの一番上にある日本語を削除

入力ソースの追加 -> ⋮ -> その他 英語(US)

入力ソースの一番上に英語(US)を移動する
